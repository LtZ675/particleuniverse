Detailed Description

Abstract (pure virtual) dependency class

Remarks
In some cases, an attribute of a ParticleSystem or its underlying components (ParticleEmitter, ...) may depend on another value that changes over time. The ´changing value´ is wrapped into a IDependency class and the attribute is ´continuesly´ changed by the IDependency subclass.
